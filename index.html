📺 Setup no vMix
Como configurar — faça isso uma vez por máquina
1
No vMix, crie 2 inputs Colour transparentes com os nomes Colour PVW e Colour PGM.
2
Crie 2 inputs GT Title (com um único campo de imagem) e nomeie como configurado abaixo (default: Multitela.gtzip PVW e Multitela.gtzip PGM).
3
Em cada Colour, adicione o respectivo Multitela.gtzip como Layer 10.
4
Configure abaixo os nomes base e o campo de imagem do GT.
Base Colour: → {base} PVW / {base} PGM
Base GT Title: → {base} PVW / {base} PGM
Campo imagem GT: → ex: Image1.Source
Pasta raiz: → caminho fixo desta máquina
